# TrueCost — Car Affordability Reality Check

TrueCost is a comprehensive financial intervention tool designed to help Malaysian car buyers understand the **actual** monthly expense of vehicle ownership. Unlike traditional loan calculators that only display monthly installments, TrueCost reveals hidden costs like fuel, insurance, maintenance, and depreciation to provide a complete financial picture.

## 🚀 Project Overview

In Malaysia, personal vehicles are often essential due to limited public transport, yet many households underestimate the true cost of ownership. This lead to financial strain and poor long-term planning. TrueCost bridges this gap by offering:

* 
**Total Cost Disclosure:** Calculates the real monthly cost (loan + fuel + insurance + road tax + maintenance + depreciation).


* 
**Personalized Affordability Assessment:** Evaluates a user's unique profile (salary, dependents, job stability) to determine a "Safe Car Budget".


* 
**Wealth Gap Analysis:** Visualizes the opportunity cost of choosing a cheaper car and investing the savings.


* 
**AI Financial Advisor:** Provides tailored advice and vehicle recommendations based on the user's profile.



## 🛠️ Technology Stack

TrueCost is built as a zero-latency, single-page application designed for easy deployment.

* 
**Frontend:** Pure HTML5, CSS3, and Vanilla JavaScript (No frameworks or build tools).


* 
**AI Inference:** Groq API using the `llama-3.3-70b-versatile` model for real-time recommendations and advisory.


* 
**Data:** A static JSON array containing a dataset of 919 Malaysian vehicles.


* 
**Visualization:** Programmatic SVG line and bar charts rendered via JavaScript.



## 📋 System Architecture

The application is structured into six progressive steps:

1. 
**Overview:** Landing page highlighting the "installment vs reality" problem.


2. 
**Your Finances:** Captures core financial data and computes safe budget limits and risk levels.


3. 
**Car Selection:** Filterable vehicle grid with AI-powered recommendations.


4. 
**True Cost Breakdown:** Detailed monthly and 5-year cost analysis.


5. 
**Scenarios (Wealth Gap):** Interactive charts showing potential wealth build-up over 5 and 10 years.


6. 
**AI Insight:** Personalized financial report and interactive AI chat advisor.
